## 기억동행 (Memory Companion) app
**어르신의 기억을 따뜻하게 기록하는 음성 기반 대화 노트 앱**

**🧭 프로젝트 소개**

복지관에서 맞춤형 돌봄 서비스(어르신 한 분 한 분의 상황에 맞춰 돌봄을 제공하는 국가 서비스)를 경험하며,
어르신을 깊이 이해하기 위해선 단순한 행정 데이터가 아니라
사전 대화 기록·감정·취향 같은 개인적 맥락이 필요하다는 걸 깨달았습니다.

이 정보가 미리 축적되어 있다면
상담도 더 빨라지고, 케어도 훨씬 따뜻해질 거라 생각합니다.

그래서 저는,
어르신의 기억과 감정을 자연스럽게 기록하고,
생활지원사가 다시 활용할 수 있으며 기록이 문서로도 남는 시스템을 만들기로 했습니다.

**🎯 핵심 가치**

말로 기록하는 쉬운 UX – 글자를 몰라도 사용 가능

감정·기억을 보존하는 기록 도구

데이터 기반 복지 케어를 돕는 관리 기능

어르신과 생활지원사, 2가지 모드 지원 (핵심 설계)

👥 사용자 모드 설계 (차별화 포인트)

**👵 어르신 모드**

큰 글자 + 최소 UI

🎙 말하면 기록됨

📖 지난 이야기 회상

😊 기분 선택 기능 (옵션)

**🧑‍🦳 생활지원사 모드**

어르신의 기록 전체 열람

감정 / 카테고리 / 기간 필터

통계 기반 케어 판단 도움

PDF 내보내기 (추가 예정)

같은 앱, 다른 사용자 경험
→ "사용자 페르소나 기반 UX 설계"로 설명 가능

**📱 주요 기능**
기능	설명
음성 인식(STT)	어르신이 말하면 자동으로 텍스트 변환
