# 기억동행 (Memory Companion)
**어르신의 기억을 따뜻하게 기록하는 음성 기반 대화 노트 앱**
말하면 남고, 쌓이면 기억이 됩니다. ☁️🌿

## 🧭 프로젝트 소개

복지관에서 **노인맞춤돌봄서비스(어르신 한 분 한 분의 상황에 맞춘 국가 돌봄 사업)** 를 경험하며,
어르신을 제대로 이해하기 위해서는 **단순한 행정 정보**만으로는 부족하다는 걸 깨달았습니다.

대화, 감정, 취향, 기억 — 이런 개인적 맥락이 사전에 기록되어 있다면
상담은 더 자연스럽고, 케어는 훨씬 따뜻해질 수 있습니다.

그래서 저는
**목소리로 이야기를 남기고 → 기록으로 축적되고 → 돌봄**에 다시 활용되는 시스템을 만들기로 했습니다.

## 🌱 프로젝트가 가진 의미

이 앱은 단순한 기록 도구를 넘어,
어르신의 삶과 감정이 ‘데이터’로 쌓이는 새로운 형태의 돌봄 도구입니다.

기록은 남고,
그 기록은 다시 돌봄이 됩니다.
